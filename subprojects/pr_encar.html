Encar 🚗

завершён HTML / CSS / JS
🐙 GitHub

Многостраничный лендинг для вымышленного сервиса по импорту автомобилей из Кореи (EncarRUS). Учебный проект по вёрстке — сделан по макету с нуля.

Цели проекта:

  • Сверстать многостраничный сайт с общей навигацией
  • Подключить и настроить слайдер изображений (Slick.js)
  • Реализовать каталог автомобилей в виде карточек
  • Добавить кастомный шрифт иконок и контактную форму
