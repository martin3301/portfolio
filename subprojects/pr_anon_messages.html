anon_messages_tg 🤖

завершён Python
🐙 GitHub

(В фотографиях приведено работаюший прототип проекта)
Telegram-бот для анонимной отправки сообщений в канал. Пользователь пишет боту — сообщение появляется в канале от случайного псевдонима. Псевдонимы ротируются, история хранится в SQLite асинхронно.

Цели проекта:

  • Разобраться с архитектурой aiogram 3.x и middleware
  • Научиться работать с асинхронными базами данных (aiosqlite)
  • Реализовать систему ротации псевдонимов без коллизий
  • Задеплоить бота и убедиться в стабильной работе в продакшене
