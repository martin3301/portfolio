Miami 🌴

завершён HTML / CSS / JS

Одностраничный лендинг для вымышленного агентства недвижимости в Майами (Condo Kings). Учебный проект по вёрстке — сделан по макету с нуля.

Цели проекта:

  • Сверстать полноэкранную hero-секцию с фоновым изображением
  • Реализовать карточки объектов недвижимости с фото и описанием
  • Добавить секцию отзывов и команды
  • Отработать CTA-элементы и структуру коммерческого лендинга
